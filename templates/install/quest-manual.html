Manual Quest install

Requirements:

  • Facebook Account
  • Quest 2
  • Almost any Windows PC/Laptop device

Device Setup

To install the custom repository for Bonelab on your Quest 2 headset we need to enable developer mode. To do this, you have to be a registered "developer" on the Meta website, it's free, quick and easy. Open https://developer.oculus.com/sign-up/ and create an 'organization'. You'll need to accept the developer agreement. Most accounts will already be verified. Next you'll need to create an organisation like above.

Joining or Creating an Organization

To create a new organization, follow these steps:

  1. Go to https://developer.oculus.com/manage/organizations/create/.
  2. Fill in the appropriate information.

Verify Your Account

  1. Go to https://developer.oculus.com/manage/verify/.
  2. If prompted, login with your Meta Developer account.
  3. Verify your account with the options presented.

Enable Developer Mode

Once you belong to a developer organization, follow these steps to put your device in developer mode:

  1. Put on the Meta headset and sign into the developer account you signed up with.
  2. On the headset, go to Settings > System > Developer, and then turn on the USB Connection Dialog option.
  3. Connect the headset to the computer using a USB-C cable and put on the headset again.
  4. Click Allow when prompted to allow access to data.
  5. Accept Allow USB Debugging and Always allow from this computer when prompted on the headset.

Install the repository

  1. Plug your headset directly into your computer if not already and accept any options for connection within your headset. Once you have done this, and your headset is powered on, with developer mode enabled you will see your Quest 2 appear within “This PC” on windows.
  2. Navigate into the Quest 2 folders by double clicking Quest 2 within “This PC” on windows.
  3. Once you are in here, follow the following folder path: Android > data > com.StressLevelZero.BONELAB > files
  4. You will see the file within here called “repositories.txt”. Right click, and copy this file.
  5. Paste this file to your desktop, by right clicking again here and pasting.
  6. Add the URLs from above to the end of file.
  7. Once you have edited the text file, drag this into your Quest “files” folder below again: Android > data > com.StressLevelZero.BONELAB > files
  8. It will ask to copy and replace. Select this option to overwrite the current file.
